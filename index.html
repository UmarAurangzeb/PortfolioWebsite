Word Correcter

"Word Corrector" is a project that demonstrates data structures and algorithms. It prompts the user to enter a paragraph and then updates the text by replacing incorrect words with the correct ones using the Levenshtein distance algorithm. The project also utilizes linked lists, hash maps and jagged array.
